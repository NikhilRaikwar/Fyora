# Fyora

Fyora is a Linktree-style creator money page where supporters can pay from any chain and creators receive on their preferred chain through Particle Universal Accounts.

Website: https://www.fyora.app/
X/Twitter: https://x.com/getfyora

## Status

This repository currently contains the polished mocked UI for the UXmaxx Hackathon demo. Real Magic + Particle Universal Accounts integration is the next implementation step.

See [FYORA_PRD.md](./FYORA_PRD.md) for the updated hackathon/product plan covering Particle Universal Accounts, Magic embedded wallets, Arbitrum settlement, and the Supabase production migration.

## Run Locally

```bash
bun install
bun run dev
```

## Demo Flow

- Claim a creator handle
- Customize a creator money page
- Open a public creator page
- Run the mocked support/payment flow
- View dashboard payments and settlement settings
